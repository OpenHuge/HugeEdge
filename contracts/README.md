# HugeEdge Contracts

Phase 0 + Phase 1 contracts define capability discovery, runtime adapters, WASM packaging for WasmEdge, optional eBPF probe profiles with a `5.15` kernel baseline, and remediation policy shapes.

PQC-hybrid transport, TUF signing, and Ed25519 verification are crypto-agility hooks at this phase. They are documented in schemas and metadata fields but are not production implementations.
